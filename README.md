## ğŸš€ Quick Start

### Prerequisites
- Python 3.10+
- pip (Python package manager)

### Installation

1. **Clone or navigate to the project folder**:
   ```bash
   cd mobile-addiction-risk-detector
   ```

2. **Create a virtual environment** (optional but recommended):
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # On Windows: .venv\Scripts\activate
   ```

3. **Install dependencies**:
   ```bash
   pip install streamlit pandas scikit-learn joblib
   ```

### Running the App

1. **Train the ML model** (one-time setup):
   ```bash
   python train_model.py
   ```
   This creates `addiction_model.pkl` that the app uses.

2. **Start the Streamlit app**:
   ```bash
   streamlit run app.py
   ```

3. **Open in browser**:
   - Local: `http://localhost:8501`
   - Network: `http://192.168.1.4:8501`

## ğŸ› Troubleshooting

### Port Already in Use
If port 8501 is in use, Streamlit will use the next available port. Check the terminal output for the correct URL.

### Model Not Found
Ensure you've run `python train_model.py` first to generate `addiction_model.pkl`

### Dependencies Error
Reinstall dependencies:
```bash
pip install --upgrade streamlit pandas scikit-learn joblib
```

### Permission Denied (macOS/Linux)
Use `python` instead of `python3`:
```bash
python train_model.py
python -m streamlit run app.py
```
